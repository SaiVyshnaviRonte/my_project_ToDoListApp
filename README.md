
# Smart To-Do List App (Console-Based)

A simple Java console application to manage tasks efficiently. Users can add tasks, view them, and mark them as completed with categories and priority levels.

## Features
- Add tasks with title, category, and priority  
- View all tasks  
- Mark tasks as completed  
- Categorization: Work, Study, Personal  
- Priority: High, Medium, Low  

## How to Run
1. Clone the repo: `git clone https://github.com/yourusername/smart-todo-list.git`  
2. Compile: `javac *.java`  
3. Run: `java ToDoListApp`
